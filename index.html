Cloudsmith 包源

第一版官方包源为 Cloudsmith OSS。当前公共仓库未配置 PyPI upstream proxy，因此安装命令默认使用 extra index。

🔌

Driver 接入

外部 driver 项目优先依赖 modlink-sdk，通过 modlink.drivers entry point 被宿主发现。

📡

统一流模型

宿主围绕 StreamDescriptor 和 FrameEnvelope 协作，避免把展示层和采集层绑定到具体设备协议。

🧩

分层架构

SDK 负责接入契约，Core 负责运行时，UI 负责页面，App 负责最终装配和启动。

ModLink Studio 文档总览 ​

ModLink Studio 的公开包源位于 Cloudsmith 仓库 xylt-space/modlink-studio。文档站说明安装方式、官方插件和 driver 开发模型；源码仓库则负责维护实现本身。

从哪里开始 ​

  • 想安装和运行：看 安装与发布

  • 想理解宿主如何组装：看 App 组合根
  • 想直接查接口：看 API 快速索引
